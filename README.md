Website Overview
This is a fully responsive, multi-page personal portfolio website built for the ITGY 401 capstone project. It demonstrates the use of HTML5, CSS3, and JavaScript to create a professional, interactive web presence.


 Pages Includes
1. Home Page: Introduction and feature highlights
2. About Page: Personal bio with skills table
3. Services Page: List of services offered
4. Contact Page: Interactive contact form with validation

Technical Features

HTML5
- Semantic HTML elements (<header>, <nav>, <section>, <footer>)
- 4 linked pages with consistent navigation
- HTML table on About page
- HTML form on Contact page

 CSS3
- External stylesheet with consistent design
- Responsive design (mobile, tablet, desktop)
- Hover effects and transitions
- Dark mode support

 JavaScript
- Form validation with error messages
- Dark mode toggle button
- Time-based greeting
- Smooth scrolling
- Console welcome message
